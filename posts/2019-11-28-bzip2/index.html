Compiling with libbzip

Nov 28, 2019 · 292 words · 2 minute read linux build

Let’s try to compile this program:

#include <bzlib.h>

int main() {
    BZ2_bzReadOpen;
}

It links against a third-party library, and references some symbol (I don’t have any idea how to actually use this library).

[I] vagrant ubuntu-bionic /v/l/c/c/bzplay ❯ g++ main.cc
[I] vagrant ubuntu-bionic /v/l/c/c/bzplay ❯

This just works. No flags needed. I was a bit confused, I would have thought surely at least a -lbz2 or something would have been needed.
